API 接口文档

RESTful API

GET/api/users - 获取用户列表
GET/api/users/:id - 获取单个用户
POST/api/users - 创建用户
PUT/api/users/:id - 更新用户
DELETE/api/users/:id - 删除用户
PATCH/api/users/:id - 部分更新用户

文件上传

POST/upload/single - 单文件上传
POST/upload/multiple - 多文件上传
GET/upload/info/:fileId - 获取文件信息
DELETE/upload/:fileId - 删除文件

分片上传

POST/chunk/init - 初始化分片上传
POST/chunk/upload - 上传分片
POST/chunk/merge - 合并分片
POST/chunk/verify - 秒传验证
GET/chunk/status/:uploadId - 获取上传状态

断点续传

POST/resume/init - 初始化上传
POST/resume/upload - 上传数据块
POST/resume/complete - 完成上传
GET/resume/status/:fileId - 获取上传状态

SSE (Server-Sent Events)

GET/sse/events - SSE 事件流
GET/sse/custom - 自定义事件流

WebSocket

WSws://localhost:3000 - WebSocket 连接
